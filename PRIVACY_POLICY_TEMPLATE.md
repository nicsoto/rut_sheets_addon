## 1. Overview

RUT Cleaner Chile processes data in Google Sheets to validate and normalize Chilean RUT values.

## 2. Data we process

1. Input values from user-selected cells in Google Sheets.
2. Derived fields generated by the add-on (normalized RUT, validation flag, error reason, duplicate flag).
3. Operational metadata (job status, processed row counts, timestamps).

## 3. Data storage

1. Processing results are written back into the active spreadsheet.
2. Lightweight job state and user preferences are stored in Apps Script Properties.
3. Optional telemetry logs can be stored in Cloud Logging and optionally in a hidden sheet if enabled.

## 4. Data sharing

We do not sell personal data and do not share sheet content with ad networks.

## 5. Security

1. We request minimum OAuth scopes required for operation.
2. Access is limited to the active spreadsheet context.

## 6. Retention

1. Spreadsheet results remain under customer control.
2. Apps Script Properties can be cleared by administrators.
